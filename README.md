# python-cli-nose2-sieve-seg

## Description
Given a number find all primes.
POC for nose2.

## Tech stack
- python
  - nose2

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
